# Contribution guidelines

Thank you for considering contributing to the extension! This document outlines some of the conventions on development workflow, testing protocol, and other resources to make it easier to get your contribution accepted.

If you have any questions, feature requests, or project ideas, please feel free to share them in our GitHub repository's Discussions or Issues.


## Development

### Getting started

To set up the development environment, follow these steps:
- Fork the repository and clone it to your local machine
- Install Node.js (optionally using mise) and npm
- Install development dependencies (npm ci)
